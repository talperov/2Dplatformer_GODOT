# 2Dplatformer_GODOT
A personal project to learn and master the Godot game engine by building a complete 2D platformer from scratch. The project covers player movement, animations, jumping mechanics, tile-based terrains, moving platforms, pickups, scene transitions, and an interactive HUD. 

# Cell Quest

A retro-style 2D platformer made in Godot 4.

Collect 4 energy cells per level to open the portal and advance to the next area.  
Avoid falling off the map â€” or you'll respawn with the level reset!

### Features
- Smooth player movement, jumping & one-way platform dropping
- 3+ levels with dynamic loading
- Energy cell collection + portal progression
- HUD showing collected cells
- Main menu (Play / Help / Quit)
- Help screen with controls
- Pause menu (Esc)
- Death & respawn system

### Controls
- Move Left: **A**  
- Move Right: **D**  
- Jump: **W** or **Space**  
- Drop through platforms: **S** or **Down Arrow**  
- Pause: **Esc**  
- Quit to Menu (while paused): **Q**

### Made With
- Godot 4
- Pixel art sprites
